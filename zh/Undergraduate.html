# 项目经历

# 倾转旋翼整体设计

完成了对于倾转旋翼无人机的整体气动外形以及具体机械结构的设计制作。

# 车顶起降Fluent分析

基于Fluent，使用SolidWorks建模，研究无人机在车顶动态起降过程中由于车顶地效及自然风等因素下受到的气流扰动，并利用机械结构解决其车顶降落问题，实现动态车顶起降。

# 共轴垂直起降无人机设计

作为小组负责人对于其整体方案以及具体结构进行设计制作。

# 语音控制四轴

制作完成了利用手机App语音控制的无人四轴飞行器，在Ubuntu16.0系统下搭建了基于Qt的px4开源飞控代码及其 QGround 地面站程序的编译环境，对其进行二次开发。使用易安卓开发语音识别APP，对飞行器进行语音及遥控器控制。
